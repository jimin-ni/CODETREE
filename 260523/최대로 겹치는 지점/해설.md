# 문제 제목

## **📝문제 설명**
1차원 직선 상에 N개의 선분이 놓여 있습니다.  
가장 많이 겹치는 곳에서는, 몇 개의 선분이 겹치는지를 구하는 프로그램을 작성해보세요.  
단, 끝점에서 닿는 경우에도 겹치는 것으로 봅니다.

### **⚠제한사항**  
2 <= N <= 100
1 <= x1 < x2 <= 100  

### **입출력 예**
입력
```js
3  
1 5  
4 6  
2 4  
```

출력
```js
3
```  

## **🧐CODE REVIEW**

### **🧾나의 풀이**

```js
const fs = require("fs");
const input = fs.readFileSync(0).toString().trim().split('\n');

const n = Number(input[0]);
const segments = [];
for (let i = 1; i <= n; i++) {
    segments.push(input[i].split(' ').map(Number));
}

// 100개의 0으로 찬 배열
const arr = [];
for (let i = 0; i <= 101; i++) {
    arr[i] = 0;
}

// N 번 반복한다
for (let i = 0; i < n; i++) {

    let start = segments[i][0]
    let end = segments[i][1]

    // arr 배열에 정해진 영역까지 +1
    for (let j = start; j <= end; j++) {
        arr[j] += 1
    }
}

// 최대로 겹치는 지점 찾기
let maxPoint = Math.max(...arr)
console.log(maxPoint)
```

#### **📝해설**
1. n 값을 정의한다
```js
const n = Number(input[0]);
```
2. segments 값을 정의한다.
   segments는 `[ [ 1, 5, 0 ], [ 4, 6 ], [ 2, 4 ] ]` 이런식의 배열이다.
   줄바꿈 표시 때문에 0이 포함되기도 한다.
```js
const segments = [];
for (let i = 1; i <= n; i++) {
    segments.push(input[i].split(' ').map(Number));
}
```
3. 100개의 0으로 찬 배열
```js
const arr = [];
for (let i = 0; i <= 101; i++) {
    arr[i] = 0;
}
```
4. n번 선분 사이의 영역에 +1을 하는 과정을 반복한다.
```js
for (let i = 0; i < n; i++) {

    let start = segments[i][0]
    let end = segments[i][1]

    for (let j = start; j <= end; j++) {
        arr[j] += 1
    }
}
```
5. 최대로 겹치는 지점을 찾아 출력한다.
   ```js
   let maxPoint = Math.max(...arr)
   console.log(maxPoint)
   ```
#### **😅개선점**

1. 0으로 찬 배열 생성을 더 간단히 하기
   segments의 순회를 끝까지하기 위해서 100보다 1 더 큰 101까지를 범위로 가진다.
   `let arr = Array(101).fill(0);`


### **다른 풀이**

```js
const fs = require("fs");
const input = fs.readFileSync(0).toString().trim().split('\n');

// 변수 선언 및 입력
const n = Number(input[0]);
const segments = [];
for (let i = 1; i <= n; i++) {
    segments.push(input[i].split(' ').map(Number));
}

const blocks = Array(101).fill(0);

// 블럭을 특정 구간에 쌓아줍니다.
segments.forEach(segment => {
    let [a, b] = segment;

    for (let i = a; i <= b; i++) {
        blocks[i] += 1;
    }
});

// 최댓값을 구합니다.
const maxNum = Math.max(...blocks);
console.log(maxNum);

```

#### **📝해설**
1. 시간복잡도는 O(NR)  
2. 공간복잡도는 O(N + R)  
3. segments 배열 안의 선분 데이터를 처음부터 끝까지 하나씩 자동으로 꺼내온다. 배열의 시작점과 끝점을 선언할 필요 없다.
   `segments.forEach(segment => { ... });`
4. 구조 분해 할당
   `let [a, b] = segment;`
   꺼내온 선분 데이터(3)에서 첫 번째 수는 a 변수에 넣고, 두 번째 수는 b변수에 넣는다. 2차원배열을 사용하지 않는다.
5. a부터 b를 포함한 범위까지 +1 진행
   ```js
   for (let i = a; i <= b; i++) {
    blocks[i] += 1;
   }
   ```
