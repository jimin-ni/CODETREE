# 문제 제목

## **📝문제 설명**
2N개의 정수가 주어졌을 때 겹치지 않으면서 2개의 원소가 하나의 그룹을 이루도록 하여 총 N개의 그룹을 만드려고 합니다. 적절하게 그룹을 만들어 각 그룹 내에 있는 2개의 수의 차이 중 최솟값이 최대가 되도록 하는 프로그램을 작성해보세요.\n

### **⚠제한사항** 
1 <= N <= 1000000 \n
1 <= 주어진 정수 <= 100000000\n

### **입출력 예**
입력:
3
15 2 7 9 10 5

### **🧾나의 풀이**

```js
function solution(topping) {
    let answer = 0;
    for(let i = 0; i < topping.length; i++) {
        const chulsu = new Set(topping.slice(0, i));
        const brother = new Set(topping.slice(i));
        if(chulsu.size === brother.size) answer++;
    }
    return answer;
}
```
