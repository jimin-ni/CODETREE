# 문제 제목

## **📝문제 설명**
2N개의 정수가 주어졌을 때 겹치지 않으면서 2개의 원소가 하나의 그룹을 이루도록 하여 총 N개의 그룹을 만드려고 합니다. 적절하게 그룹을 만들어 각 그룹 내에 있는 2개의 수의 차이 중 최솟값이 최대가 되도록 하는 프로그램을 작성해보세요.<br>

### **⚠제한사항** 
1 <= N <= 1000000  <br>
1 <= 주어진 정수 <= 100000000 <br>

### **입출력 예**
입력:<br>
3<br>
15 2 7 9 10 5<br>
<br>
출력: 5 <br>

## **🧐CODE REVIEW**

### **😫나의 오답 풀이**

### **🧾나의 풀이**

```js
const fs = require("fs");
const input = fs.readFileSync(0).toString().trim().split('\n');
const n = Number(input[0]);
const arr = input[1].trim().split(' ').map(Number);

let minInGroup = Infinity
let maxNum = 0;
for (let i = 0; i < 2n; i++) {
    let x = arr[i]
    for (let j = i + 1; j < 2n; j++) {
        let y = arr[j]
        let xy = Math.abs(x - y)
        minInGroup = Math.min(minInGroup, xy)
    }
    maxNum = Math.max(maxNum, minInGroup)
}
console.log(maxNum)
```

### **🔖정리**

1. 짝을 지어 묶은 수 간의 최소값을 여러 세트 만들고, 그 여러 세트 간의 최대값을 구하는 게 문제이다.<br>
2. 만약, 주어진 수가 `15, 2, 7, 9, 10, 5`이고, 오름차순 정렬을 한다면 `2, 5, 7, 9, 10, 15` 이렇게 된다.<br>
3. i, i + n 끼리 짝이 될것이고,<br>
    * 1번 그룹: (2, 9) ➔ 차이는 7<br>
    * 2번 그룹: (5, 10) ➔ 차이는 5<br>
    * 3번 그룹: (7, 15) ➔ 차이는 8<br>
4. 이 3개의 차이 값 중에서 가장 작은 값(최솟값)은 5.<br>
5. 만약 오름차순 정렬이 아니라 제일 작은 수와 제일 큰 수를 묶는 식이라면,<br>
    * (2, 15) ➔ 차이는 13<br>
    * (5, 7) ➔ 차이는 2<br>
    * (9, 10) ➔ 차이는 1<br>
6. 이 세 값 중에서 제일 작은 값은 1이다.<br>
7. 때문에, 모든 그룹이 골고루 큰 차이를 가지게 하면 최솟값 자체가 커질꺼고, 그럼 문제가 원하는 최댓값을 구할 수 있다.<br>
