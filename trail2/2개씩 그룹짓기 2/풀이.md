# 문제 제목

## **📝문제 설명**
2N개의 정수가 주어졌을 때 겹치지 않으면서 2개의 원소가 하나의 그룹을 이루도록 하여 총 N개의 그룹을 만드려고 합니다. 적절하게 그룹을 만들어 각 그룹 내에 있는 2개의 수의 차이 중 최솟값이 최대가 되도록 하는 프로그램을 작성해보세요.<br>

### **⚠제한사항** 
1 <= N <= 1000000  <br>
1 <= 주어진 정수 <= 100000000 <br>

### **입출력 예**
입력:<br>
3<br>
15 2 7 9 10 5<br>
<br>
출력: 5 <br>

## **🧐CODE REVIEW**

### **😫나의 오답 풀이**

### **🧾나의 풀이**

```js
const fs = require("fs");
const input = fs.readFileSync(0).toString().trim().split('\n');
const n = Number(input[0]);
const arr = input[1].trim().split(' ').map(Number);

let minInGroup = Infinity
let maxNum = 0;
for (let i = 0; i < 2n; i++) {
    let x = arr[i]
    for (let j = i + 1; j < 2n; j++) {
        let y = arr[j]
        let xy = Math.abs(x - y)
        minInGroup = Math.min(minInGroup, xy)
    }
    maxNum = Math.max(maxNum, minInGroup)
}
console.log(maxNum)
```

#### **😅해설**

1. 배열을 오름차순으로 정렬한다. <br>
단, 중요한 점은 그냥 `sort()` 메서드를 사용하는 것이 아니라 `.sort((a, b) => a - b)` 를 입력한다.  <br>
 <br>
2. 총 주어진 수는 2n개이지만, 오름차순으로 정리한 수의 절반을 나눠 서로 매칭시키면 두 수 차이 중 최소값을 구할 수 있다. <br>
    `for(let i = 0; i < n; i++){}` <br>
<br>
3. 두 수의 차이를 diff에 담는다. ` diff = arr[i+n] - arr[i]`<br>
<br>
4. 그중에서 가장 작은 값을 고른다. `minDiff = Math.min(minDiff, diff)` <br>
<br>
5. 답을 출력한다. `console.log(minDiff)` <br>
